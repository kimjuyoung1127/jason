# Jason Portfolio Landing PRD

> Version: 0.1.0
> Last Updated: 2026-02-20
> Owner: Jason

## 1. 목표
- 외주 고객에게 신뢰와 실력을 즉시 전달하는 포트폴리오 랜딩페이지를 구축한다.
- Jason의 다양한 코딩 프로젝트를 감각적으로 소개하고 문의 전환을 높인다.

## 2. 디자인 원칙
- 색상: 블랙/차콜/화이트 + 레드 포인트, 보라색 금지.
- 톤: 에디토리얼 + 실험적 인터랙션, 과한 장식보다 의도적 강약.
- 레퍼런스: `referecnce.md`의 타이포/노이즈/레이어드 레이아웃 문법을 계승.

## 3. 핵심 기능
- 메인 히어로: 아래는 좁고 위로 갈수록 넓어지는 3D 스파이럴 카드 홀더.
- 인터랙션: 카드 선택 시 `/projects/[slug]` 디테일 페이지로 이동.
- 반응형: 모바일에서는 성능 우선, 효과 강도 자동 축소.

## 4. 정보 구조
- `/`: 랜딩 (히어로, 소개, 프로젝트 프리뷰)
- `/projects/[slug]`: 프로젝트 상세

## 5. API 초안
- `GET /projects`
- `GET /projects/:slug`
- `POST /contact`

## 6. 성능 목표
- 초기 뷰는 2D 콘텐츠 우선 렌더링 후 3D 로딩.
- 3D 컴포넌트는 동적 import로 분리.
- 모바일에서 프레임 유지가 어려우면 카드 수/애니메이션 빈도를 감소.

## 7. 폴더 정책
- `apps/frontend` ↔ `apps/backend` 모듈명 1:1 유지.
- 모든 작업 기록은 `archive/`에 누적해 RAG 입력으로 사용.

## 8. 수용 기준
- 히어로에서 스파이럴 구조가 시각적으로 명확하다.
- 카드 클릭 시 상세 라우트가 정상 동작한다.
- 모바일/데스크톱에서 레이아웃 깨짐이 없다.
- 보라색 계열 색상이 포함되지 않는다.
