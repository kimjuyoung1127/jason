# 이 폴더는 프론트엔드 React 컴포넌트를 분류별로 포함합니다.

## 하위 폴더
- `hero/` — 3D 스파이럴 히어로 및 관련 컴포넌트
- `ui/` — 공통 재사용 UI 컴포넌트 (버튼, 오버레이, 커서 등)
- `sections/` — 랜딩페이지 전환 섹션 (Services, FAQ, CTA 등)
- `layout/` — 페이지 레이아웃 컴포넌트 (Header)

## 네이밍 규칙
- 컴포넌트: PascalCase (예: SpiralHero)
- 파일명: kebab-case (예: spiral-hero.tsx)
- CSS Module: 컴포넌트와 동일 이름 (예: spiral-hero.module.css)
- 모든 파일 첫 줄에 기능 설명 주석 필수
